# ChatGPT Prompt PopClip Extension

无弹窗版本：划词后点击图标，直接跳转 ChatGPT 并自动填充模板。

## 用法

1. 下载并安装 `ChatGPTPrompt.popclipextz`。
2. 在任意应用中划词，点击 ChatGPT 图标。
3. ChatGPT 会预填以下内容：

```text
【原文】
<选中文本>

【问题】
```

4. 在 `【问题】` 下自行输入，再手动发送。

## 行为说明

- 不再弹输入框。
- 不自动发送。
- 同时复制模板到剪贴板，作为兜底。
- 在 PopClip 设置窗口中禁用触发（避免误触）。
