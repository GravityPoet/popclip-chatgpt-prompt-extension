# ChatGPT Prompt PopClip Extension

独立仓库：ChatGPT Prompt PopClip 扩展。

## 功能

- 划词后点击 ChatGPT 图标
- 先弹窗补充问题
- 自动拼接“【原文】+【问题】”
- 通过 `https://chatgpt.com/?q=` 直发

## 相比旧方案优势

- 比仅发送原文更清晰：发送前补问一句
- 结构化提示词更稳定
- 保持直发流程，不增加复制步骤

## 安装

1. 打开 Releases 页面下载 `ChatGPTPrompt.popclipextz`
2. 双击安装到 PopClip
